High Concept

"Billiards meets Samurai"

Ricochet Ronin is a high-speed, top-down physics action game. The player controls a samurai who cannot run or walk normally. Instead, the samurai slingshots themselves at high velocity like a billiard ball to slice through enemies, ricocheting off walls to maintain momentum and compile combos.

Core Gameplay Loop

The rhythm of the gameplay is defined as: Still (Aim) → Motion (Dash) → Still (Aim).

1. Aim Phase

Hold mouse/touch to slow down time. Drag in the opposite direction (slingshot mechanical action). The system computes a 2-bounce trajectory prediction line using vector reflections.

2. Dash Phase

Release mouse to launch at high speed. The samurai slides, ricocheting off wall colliders using a physics material. Passing through enemies slashes them instantly.

3. Reset Phase

As friction slows the player down below a certain threshold, the player stops and enters the Idle state, preparing for the next slingshot dash.

Game Core Variables

  • Player Life Time: Serves as the player's health. Decreases in real-time (drains 3x faster during slow-motion aiming). Killing enemies rewards seconds, while taking damage or using active mutations penalizes seconds.
  • Level Survival Time: Count-down timer for each floor. To complete a floor, the player must survive the timer AND defeat all spawned enemies.
  • Combo Counter: Tracks the number of enemies killed within a single dash or short window to reward significant life time bonuses and high combo alerts.
